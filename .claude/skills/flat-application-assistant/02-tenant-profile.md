## Mapping to Listing Language

When a listing or landlord message uses this language, it signals a **good fit** for this persona:
- "unbefristet", "ruhige Lage", "Familien willkommen" / "Berufstätige bevorzugt"
- Landlord explicitly lists required documents (signals an organized, serious process worth investing effort in)

When a listing uses this language, flag as **friction, not necessarily a deal-breaker**:
- "nur an WG" when the household isn't a WG
- "Tierhaltung nicht erlaubt" if a pet is part of the household
- "ab sofort" when the actual move-in date is weeks out (address directly in the Anschreiben rather than ignoring it)
