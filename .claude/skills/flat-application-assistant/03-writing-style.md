# Writing Style Guide (Anschreiben)

## Critical Rules

1. **NO em-dashes (--).** Use commas, periods, or restructure the sentence instead.
2. **NO rental clichés or filler phrases.** Cut: "Wir sind eine ruhige und zuverlässige Familie", "Wir würden uns riesig freuen", "Ihre Wohnung hat es uns sofort angetan", "Wir passen perfekt zu Ihrer Wohnung". These appear in nearly every generic inquiry and signal a copy-paste message.
3. **NO unverified claims about the listing.** Anything said about the flat (layout, fittings, neighborhood, view) must come directly from the listing text or photos provided by the user. Never invent a detail to sound personal.
4. **NO apologetic or over-eager language.** Not "Wir hoffen, dass Sie uns eine Chance geben" but a plain statement of fit: "Mit einem unbefristeten Arbeitsvertrag und einem Nettoeinkommen von X € erfüllen wir die üblichen Bonitätsanforderungen."
5. **Always formal "Sie".** Never "du", even if the listing itself is written casually.
6. **One specific detail from the listing, always.** Every Anschreiben must reference at least one concrete detail from this specific listing (layout, fittings, location feature, or something the landlord wrote in the ad). This is the single biggest differentiator from the 50-200 generic inquiries a popular listing receives in its first hour.
7. **Never fabricate documents or facts.** If the Selbstauskunft says no Schufa is available yet, the Anschreiben must not claim one is attached.

## Tone
- **Warm but plain.** Friendly without being effusive. A landlord skimming 100 messages responds to clarity, not enthusiasm.
- **Confident, not needy.** State facts (income, contract type, move date) rather than asking to be considered favorably.
- **Short.** Landlords decide who gets a viewing slot in seconds. Every sentence should carry information they need to make that decision.

## Structure of the Anschreiben

### Opening (1-2 sentences)
- State which listing this is about and where it was found (portal + short reference, e.g. "Ihr Inserat für die 2-Zimmer-Wohnung in der Musterstraße auf Kleinanzeigen").
- Reference the one specific listing detail immediately - not in paragraph three.

### Who is asking (2-3 sentences)
- Household composition, employment situation, and the concrete reason for the move: new job at [NEW_EMPLOYER] in Sankt Leon-Rot starting [DATE].
- State the desired move-in date plainly.

### Why this fits / fit signals (bullet list, 3-5 items)
- Net income relative to rent
- Contract type (unbefristet)
- Clean rental history / no arrears
- Household facts relevant to the listing (non-smoker, no pets / pet details if the listing allows pets)
- Availability for a short-notice viewing

### Attachments line (1 sentence)
- State what is attached or available on request: Selbstauskunft, Einkommensnachweise, Schufa, Mietschuldenfreiheitsbescheinigung. Only list what actually exists per `01-renter-profile.md` - never claim a document is attached if it isn't ready.

### Closing (1-2 sentences)
- Express availability for a viewing, give a way to reach you quickly (phone preferred for fast-moving listings), and a plain closing line. No begging.

## Bullet Point Style
- Start with a bold label: "**Einkommen:**", "**Vertragsart:**", "**Verfügbarkeit:**"
- Be specific: numbers, dates, contract terms - not adjectives
- Keep each bullet to one line where possible

## Language
- Default to German, formal register, regardless of the household's native language.
- If the listing itself is written in English (common for international landlords near Heidelberg/Mannheim, e.g. university or SAP-adjacent housing), mirror that and write the Anschreiben in English instead - note the choice to the user before drafting.

## Length
- Target 120-180 words of body text. This is shorter than a job cover letter - rental inquiries are read in seconds, not minutes, and need to be skimmable on a phone.
- Hard maximum: fits on one page with the bullet list and signature.
